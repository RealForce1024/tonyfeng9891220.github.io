Merge k Sorted Lists

思路：

  1. 每个链表的头结点入小根堆
  2. 取堆顶元素，接入newHead之后，将该元素的下一个结点入堆
  3. 循环第二步直到堆空

Convert Sorted List to Binary Search Tree

思路：

  1. 找中点
  2. 左半部分生成BST
  3. 右半部分生成BST
  4. 循环1~3

优化找中点操作为顺序遍历链表操作，挨个生成树的结点

Insertion Sort List

思路：

插入排序：每次将一个元素插入到已排好序的序列中的相应位置
