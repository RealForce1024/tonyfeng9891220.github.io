```golang

```
和使用new(Struct)是一样的

```golang
	node := new(common.LinkNode)
	node2 := &common.LinkNode{}
	node3 := common.NewNode()
	fmt.Println(node, node2, node3)
	fmt.Printf("%p,%p,%p\n", node, node2, node3)
// &{0,<nil>} &{0,<nil>} &{0,<nil>}
```
