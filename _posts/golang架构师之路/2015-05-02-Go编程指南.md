## 包 
- 每个Go程序都是由包构成
- 程序入口在`main`包下
- 该示例程序`导入`了`fmt`和`math/rand`两个`路径`。
- 根据Go语言的惯例，`包名和导入路径的最后一个元素相同`。例如，"math/rand"包含在一个以`package rand`开头的文件。

```go
package main

import(
    "fmt"
    "math/rand"
)

func main(){
    fmt.Println("My favorite number is",rand.Intn(10))
}
```
