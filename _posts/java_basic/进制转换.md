## 权重的概念
十进制 123  
权重为10 
`1*10^2+2*10^1+3*10^0 = 100+20+3 = 123`



1010表示的十进制呢，则要看其权重2  
`1*2^3+0*2^2+1*2^1+0*2^0 = 2^3+2^1 = 10`



12(8进制)表示的十进制，则要看其权重8

`1*8^1+2*8^0=8+2=10`
